# Layout Variety Reference

Use this with the screenshots in `references/style and layout/` when building decks over 20 slides or when an early contact sheet looks repetitive.

## How To Use

- Inspect the screenshot folder before coding a large deck.
- Make a slide-format plan first: each slide gets one content role and one layout family.
- Rotate families across each 5-slide run. Do not let more than two adjacent slides use the same family.
- Keep one visual language for the deck, but vary composition, density, rhythm, and text-image relationship.
- Borrow structure, not content. The screenshots are layout references, not templates to copy verbatim.

## Layout Families

- Editorial opener: oversized bilingual headline, small proof points, one calm image or abstract panel.
- Concept card: strong title, compact explanation block, one diagram or visual metaphor.
- 2x2 framework: four labeled quadrants or roles with balanced whitespace.
- Horizontal process: three to five steps across the lower or middle band.
- Vertical stack: side rail of cards, decisions, or examples.
- Comparison: left/right or before/after contrast with clear asymmetry.
- Method canvas: table-like structure with rows, labels, and examples.
- Evidence + scene: image-led slide with a small structured evidence panel.
- Big-photo method: large image plus numbered actions or checkpoints.
- Quote/reflection: expressive headline, minimal supporting copy, one restrained visual anchor.
- Agenda/list: numbered sequence with generous spacing and thin dividers.
- Loop/system map: nodes, arrows, feedback, or workflow circles with minimal text.
- Activity/instruction: clear task, time box, and participant actions.
- Closing synthesis: one final principle supported by a simple diagram or image.

## Big Deck Rhythm

For decks over 20 slides:

- Start each section with an opener or statement slide.
- Follow with 2-4 content slides using different families.
- Insert an activity, recap, quote, or case slide every 6-8 slides.
- Alternate dense teaching slides with lighter visual slides.
- Use repeated motifs intentionally: same system map can return as a progress marker, but the surrounding layout should change.
- In the final contact sheet, scan for identical silhouettes: same title position, same image box, same three-card row, or same text weight repeated too often.
